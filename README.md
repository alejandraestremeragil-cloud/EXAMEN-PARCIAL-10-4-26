# 📚 Sistema de Gestión de Biblioteca

Este es un proyecto académico desarrollado en Python que permite gestionar el inventario de una biblioteca mediante una interfaz de consola (menú interactivo).

## 🚀 Funcionalidades
* **Agregar libros**: Registra el título, autor y unidades disponibles.
* **Prestar libros**: Reduce el stock disponible tras validar que existan unidades.
* **Devolver libros**: Incrementa el stock y permite registrar libros nuevos.
* **Consultar**: Muestra la información detallada y el estado actual de un libro.

## 🛠️ Requisitos
* Tener instalado **Python 3.x**.
